Why We Use the Normalize CSS Script

Normal CSS allows for simplicity and cross-compatibility throughout all modern web browsers. It is a modular method that can be built upon from an already exisiting style. There is less CSS coding to produce through normalization than a CSS reset and it produces a smaller file size.

What is Semantic Markup?

Semantic markups are the separation of content and presentation in HTML codes. It is a way of identifying what the elements are (giving them meaning) while not explicitly identifying the content associated with the markups.
